================================================================================
                            KEY FEATURES
================================================================================

UCI PROTOCOL COMPLIANCE:
  • Full UCI protocol implementation
  • Compatible with Arena, ChessBase, Lichess, and other chess GUIs
  • Commands: uci, isready, setoption, ucinewgame, position, go, quit

SEARCH ALGORITHM:
  • Minimax with alpha-beta pruning (fundamental game theory)
  • Move ordering (captures > checks > quiet moves)
  • Transposition table (position caching)
  • Iterative deepening (time management)
  • 2-3x faster than naive minimax due to pruning

EVALUATION:
  • Material counting (standard piece values)
  • Piece-square tables (positional understanding)
  • Pawn structure analysis (doubled pawn penalty)
  • Mobility bonus (more moves = better position)
  • Endgame vs Midgame adaptation
  • Check/checkmate detection

INTERACTIVE FEATURES:
  • Play against the engine
  • Analyze positions with depth analysis
  • Algebraic notation support (e4, Nf3, etc.)
  • Position visualization
  • Undo functionality

chess_engine.py (181 lines)
  Main UCI protocol handler. Parses commands from chess GUIs, manages board
  state, and coordinates search. Implements all required UCI commands.

  Position evaluation engine. Implements material counting, piece-square tables,
  pawn structure analysis, and mobility bonuses. Returns numerical score.
